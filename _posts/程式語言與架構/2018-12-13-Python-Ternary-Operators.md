在 Jave 或 JavaScript 中的三元運算子是
```shell
判斷式 ？ 若判斷為真執行區塊 ： 若判斷為假執行區塊
```
但這不適用於 Python。

<!--more-->
<br> 

在 Python 中沒有 `?:` ，而是類似 `if...else` 的寫法

```shell
若判斷為真執行區塊 if 判斷式 else 若判斷為假執行區塊
```
